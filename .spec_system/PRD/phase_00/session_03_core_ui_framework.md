# Session 03: Core UI Framework

**Session ID**: `phase00-session03-core_ui_framework`
**Status**: Not Started
**Estimated Tasks**: ~15
**Estimated Duration**: 2-3 hours

---

## Objective

Create the responsive UI foundation including layout components, navigation, styling system, and reusable components that will be used throughout the application.

---

## Scope

### In Scope (MVP)
- Base layout component (header, main, footer)
- Navigation/menu structure with "Our Talents" link
- Responsive grid system
- CSS architecture (CSS Modules, Tailwind, or styled-components)
- Color palette and typography (aligned with actinginstitute.ma branding)
- Common UI components:
  - Button variants
  - Form inputs (text, select, checkbox)
  - Card component (for talent display)
  - Modal/dialog component
  - Loading states
- Mobile-first responsive breakpoints
- Basic page templates (home, list, detail, form)

### Out of Scope
- Talent-specific components (Session 05)
- Admin-specific layouts (Session 06)
- Advanced animations/transitions

---

## Prerequisites

- [ ] Session 01 completed (frontend framework running)
- [ ] Design direction approved (reference sites reviewed)

---

## Deliverables

1. Layout component with responsive behavior
2. Navigation component with menu structure
3. UI component library (buttons, inputs, cards, modals)
4. Styling system with design tokens (colors, spacing, typography)
5. Page templates demonstrating responsive behavior

---

## Success Criteria

- [ ] Layout renders correctly on mobile (375px) and desktop (1440px)
- [ ] Navigation is accessible and keyboard-navigable
- [ ] UI components are reusable and consistently styled
- [ ] Design aligns with reference sites aesthetic
- [ ] No horizontal scroll on any viewport size
